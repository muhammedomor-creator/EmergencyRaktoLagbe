## 📌 প্ল্যাটফর্ম পরিচিতি (About the Platform)
**"ইমারজেন্সি রক্ত লাগবে ?"** হলো একটি সম্পূর্ণ ব্যক্তিগত উদ্যোগে তৈরি ও পরিচালিত অলাভজনক এবং সম্পূর্ণ ফ্রি রক্তদান প্ল্যাটফর্ম। জরুরি মুহূর্তে কোনো প্রকার দালাল বা বাণিজ্যিক মধ্যস্থতাকারীদের হস্তক্ষেপ এবং আর্থিক হয়রানি ছাড়াই সরাসরি মুমূর্ষু রোগীর পরিবার ও রক্তদাতাদের মাঝে দ্রুততম সময়ে নিরাপদ যোগাযোগ স্থাপনের লক্ষ্যে এই ওয়েব অ্যাপ্লিকেশনটি ডিজাইন করা হয়েছে।
এই প্ল্যাটফর্মটি সম্পূর্ণ গতিশীল (Dynamic) এবং এর ডেটা ফিল্টারিং ইঞ্জিনটি বাংলাদেশের প্রশাসনিক কাঠামো (বিভাগ, জেলা এবং উপজেলা) অনুযায়ী নিখুঁতভাবে আপনার স্থানীয় এলাকায় উপযুক্ত রক্তদাতা খুঁজতে সক্ষম।
🔗 **লাইভ প্রজেক্ট লিংক:** https://emergencyroktolagbe.vercel.app
